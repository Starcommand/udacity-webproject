Intro to Programming Notes

What is the World Wide Web?

The web was invented in the early 1990s. It is a collection of HTML documents that allows website creation. HTML supports links to other documents, as well as graphics, audio, and video files. This means you can jump from page to page which is considered Hyperlink. Here's a video that gives a good analogy of what the World Wide Web is.

HTML

HTML stands for Hypertext Markup Language. It is a standard markup language for creating Web pages. HTML documents contain text content which describes "what you see" and markup describes "what it looks like". Here's a video that gives a good explanation of what HTML is.

Tags and Elements

HTML Tag is an opening or closing entity. For example: <p> and </p>

HTML Element encompasses opening tag, closing tag and content. For example: <p>I love programming</p>

Components of The Web

Components of the web consists of the browser, internet, HTTP, and servers.

  • Browser: A program that runs on your computer to display files found on the web.
  • Internet: A global network connecting millions of computers.
  • HTTP: HyperText Transfer Protocol is a simple protocol that your browser uses to communicate with servers.
  • Server: They are computers that are optimized for hosting files.

Why are Computers Stupid?

Computers are often described as being stupid by programmers because it is unambiguous when given instructions. Since the computer interprets language literally the programmer will likely run into problems.

Inline vs Block Elements

HTML elements can either be a block element or inline element. A block element creates an "invisible box" around the content. And an inline element does not start on a new line. It only takes as much width it requires.
